    }

Pick a type

  • Pointer, Buffer: 🚫

[B advantages

  • By far most common type
  • Easy to use, e.g. serializers

ByteBuffer advantages

  • Supported API for "direct" allocation
  • Cheap slicing, "views" over memory

Asymmetry

ByteBuffer → byte array: fast!

byte array ← ByteBuffer: slow!

Conclusion

  • Expose everything
  • In-place API:
    • Takes buffers only
  • Functional:
